# Exercice 1 

## taches a effectué

1. Déclarer 2 variables de type "chaine de caracteres" nommée `nom` et `prenom`. 
2. Affecter votre nom à la variable correspondante.
3. Affecter votre prenom à la variable correspondante.
4. Afficher dans la console le texte suivant : 

```
Bonjour, je m'appel {nom} {prenom}. 
```
